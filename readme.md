# Volt

The command center for your computer.

Volt is a fast, keyboard-first launcher that unifies everything you do — apps, files, commands, and workflows — into a single interface.

---

## Why Volt exists

Computers were not designed for how we work today.

We constantly switch between tools, search in multiple places, and lose time navigating interfaces built for the mouse.

Volt replaces that with a single entry point — fast, predictable, and entirely driven by the keyboard.

---

## What Volt does

Volt lets you:

- Launch anything instantly  
- Search across apps, files, and data  
- Execute commands and workflows  
- Use built-in tools (calculator, clipboard, snippets, timers)  
- Extend behavior with plugins and extensions  

All from one place.

---

## How it works

- **Native performance** — Rust backend, instant response time  
- **Keyboard-first** — no context switching, no friction  
- **Extensible system** — plugins and external extensions  
- **Cross-platform** — consistent experience across environments  

---

## The vision

Volt is not just a launcher.

It is a layer on top of your operating system —  
a faster interface between you and everything you do.
